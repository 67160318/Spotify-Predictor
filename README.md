# 🎵 Spotify Track Popularity Pro Predictor

**Spotify Predictor Pro** คือเว็บแอปพลิเคชันที่ใช้เทคโนโลยี Machine Learning ในการทำนาย **"คะแนนความนิยม (Popularity Score)"** ของเพลงบนแพลตฟอร์ม Spotify โดยวิเคราะห์จากองค์ประกอบเสียง (Audio Features) พร้อมทั้งมีระบบแนะนำเพลงที่มีลักษณะเสียงใกล้เคียงกัน (Song Recommendation)

แอปพลิเคชันนี้ออกแบบมาให้มี User Interface (UI) ในสไตล์ดาร์กโหมด (Spotify Theme) ที่ใช้งานง่าย ไม่ว่าผู้ใช้จะมีความรู้ด้าน Machine Learning หรือไม่ ก็สามารถเข้าใจและเพลิดเพลินไปกับการค้นหาเพลงใหม่ๆ ได้ทันที

🔗 **[ทดลองใช้งานแอปพลิเคชันคลิกที่นี่](https://spotify-predictor-boyrlk3od5wnzql9xrkmip.streamlit.app/)**

## ✨ ฟีเจอร์เด่น (Features)
แอปพลิเคชันนี้ถูกพัฒนาขึ้นโดยคำนึงถึงประสบการณ์ของผู้ใช้ (UX/UI) และความแม่นยำของการวิเคราะห์:
- **User-Friendly UI:** ดีไซน์สไตล์ Spotify ใช้งานง่ายด้วยแถบเลื่อน (Sliders) สำหรับปรับแต่งองค์ประกอบเสียง
- **Popularity Score:** แสดงผลการทำนายความฮิตของเพลง (0-100 คะแนน) จากค่าตัวแปรที่ผู้ใช้กำหนด 
- **Song Recommendation:** ระบบแนะนำ 5 เพลงที่มีองค์ประกอบเสียงใกล้เคียงกับค่าที่ปรับแต่งมากที่สุด (Nearest Neighbors) โดยมีการคัดกรองเพลงซ้ำออกเพื่อผลลัพธ์ที่หลากหลาย
- **Model Selection:** ผู้ใช้สามารถเลือกสลับใช้อัลกอริทึมทำนายได้ถึง 4 แบบ (Linear Regression, Decision Tree, Random Forest, KNN) เพื่อเปรียบเทียบผลลัพธ์

## 🎛️ คำอธิบายตัวแปร (Audio Features)
โมเดลของเราใช้ข้อมูลตัวแปรหลัก 9 ตัวในการวิเคราะห์และทำนายผล ดังนี้:
1. **Danceability (ความเหมาะในการเต้น):** (0.0 - 1.0) ยิ่งค่ามาก จังหวะยิ่งชัดเจนและเต้นตามได้ง่าย
2. **Energy (ความเร้าใจ):** (0.0 - 1.0) พลังของเพลง ค่าที่สูงจะรู้สึกถึงความเร็ว ดัง และหนักหน่วง
3. **Loudness (ความดัง):** (dB) ความดังเบาโดยรวมของเพลง (มักติดลบ) ยิ่งใกล้ 0 แปลว่ายิ่งดังมาก
4. **Speechiness (เสียงพูด):** (0.0 - 1.0) สัดส่วนของเสียงพูดในเพลง ค่าสูงมักจะเป็นเพลงแร็ปหรือพอดแคสต์
5. **Acousticness (ความเป็นอคูสติก):** (0.0 - 1.0) ยิ่งใกล้ 1 แปลว่าใช้เครื่องดนตรีสดสูง
6. **Instrumentalness (เพลงบรรเลง):** (0.0 - 1.0) การไม่มีเสียงร้อง ยิ่งใกล้ 1 แปลว่าเป็นเพลงบรรเลงล้วน
7. **Liveness (การแสดงสด):** (0.0 - 1.0) ยิ่งค่าสูง แปลว่าเพลงถูกบันทึกเสียงจากการเล่นสด (มีเสียงคนดู)
8. **Valence (อารมณ์ของเพลง):** (0.0 - 1.0) ค่าสูง = ร่าเริง/มีความสุข, ค่าต่ำ = เศร้า/หดหู่
9. **Tempo (ความเร็วจังหวะ):** (BPM) ความเร็วของจังหวะเพลง (Beats Per Minute)
